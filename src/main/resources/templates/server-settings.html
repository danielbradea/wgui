Server Settings

The interface addresses represent the IP addresses assigned to the WireGuard VPN server.
The listen port is the network port that the WireGuard VPN server will use to accept incoming connections.
The post up script is a script or command that is executed on the WireGuard VPN server after a client successfully establishes a connection.
The post down script is a script or command that is executed on the WireGuard VPN server after a client connection is closed or terminated.
The pre up script is a script or command that is executed on the WireGuard VPN server before a client establishes a connection.
The pre down script is a script or command that is executed on the WireGuard VPN server before a client connection is closed or terminated.
